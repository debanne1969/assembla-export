<h1 id="dns_information_for_domain_filesender.org">DNS information for domain filesender.org</h1>

<p><em>Responsible team member: Jan Meijer</em></p>

<p>Primary DNS server is UNINETT&#39;s biff.uninett.no.&nbsp; Mail is routed via UNINETT&#39;s anti-spam service &#39;dike.uninett.no&#39;.&nbsp;</p>

<pre>
&gt; host -t soa filesender.org
filesender.org has SOA record biff.uninett.no. hostmaster.uninett.no. 2010050300 14400 3600 604800 86400
</pre>

<pre>
&gt; host -t ns filesender.org
filesender.org name server biff.uninett.no.
filesender.org name server nn.uninett.no.
filesender.org name server ns.hyp.net.</pre>

<pre>
&gt; host -t mx filesender.org
filesender.org mail is handled by 10 dike.uninett.no.</pre>
